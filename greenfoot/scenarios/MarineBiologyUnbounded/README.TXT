PROJECT TITLE:
Marine Biology Simulation

PURPOSE OF PROJECT:
This is the unbounded-world Marine Biology Simulation Case Study for the
College Board AP Exam. It simulates fish in an unbounded environment.
 
VERSION or DATE:
1.0

HOW TO START THIS PROJECT:
Compile all the classes and hit the Run button.

AUTHORS:
Adapted by C.Vargas, P.Henriksen, and M.Kolling from the original
College Board case study.

USER INSTRUCTIONS:
To populate the world, select the Fish class and then hold Shift as you click on
the grid cells, or invoke one of the populate methods on the World object.
Right-click the label BoundedEnv above the grid to get the list of methods
of this object.
