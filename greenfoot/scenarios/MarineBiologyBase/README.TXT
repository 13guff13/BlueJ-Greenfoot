PROJECT TITLE:
Marine Biology Simulation

PURPOSE OF PROJECT:
This is the bounded-world Marine Biology Simulation Case Study for the
College Board AP Exam. It simulates fish in a bounded environment.
 
VERSION or DATE:
1.0

HOW TO START THIS PROJECT:
Compile all the classes and hit the Run button.

AUTHORS:
Adapted by C.Vargas, P.Henriksen, and M.Kolling from the original
College Board case study.

USER INSTRUCTIONS:
To add more fish, select the Fish class and then hold Shift as you click on
the grid cells, or invoke the populate method on the World object.
To place fish all along the boundary of the world, call the method 
populateBoundaryFish() on the World object. Right-click the label
BoundedEnv above the grid to get the list of methods of this object.
To create an empty world of a specific size, right-click the BoundedEnv
class and invoke the constructor with two arguments.
