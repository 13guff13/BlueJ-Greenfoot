PROJECT TITLE: 
Asteroids

PURPOSE OF PROJECT: 
To shoot asteroids into pieces

VERSION or DATE:
1.0

HOW TO START THIS PROJECT:
Add some asteroids into the world and one rocket

AUTHORS:
Poul Henriksen

USER INSTRUCTIONS:
Use the arrow keys to control the rocket:
 * up: accelerate
 * left: rotate rocket left
 * right: rotate rocket right
 * space: fire the gun
