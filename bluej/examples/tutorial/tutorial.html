BlueJ Tutorial

Welcome! This tutorial will help you to get started with using BlueJ. The tutorial is split into many small parts. This tutorial does not teach Java programming (whole textbooks are written on that), but rather just shows you the useful features of BlueJ:

  1. Creating objects and calling methods.
  2. Writing and reading text to the terminal.
  3. Editing and compiling code.
  4. Running code using the codepad.
  5. Creating and running tests.
  6. Teamwork using version control.
  7. GUI programming using JavaFX.
  8. The main method and running outside BlueJ.
