IMPORTANT: 
When adding a new translation, it should be added in the build script: package/build.xml
It should also be added to bluej.defs so users can see the available languages.

IMPORTANT: 
Translations need to be done using Latin-1 encoding Unicode escape sequences (encoding that is 
used by Java's .properties files, and also the output encoding from native2ascii)
